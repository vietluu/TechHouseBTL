# Dev setup
* Install [Git](http://www.git-scm.com/)
* Install [IIS](https://www.iis.net/downloads)
* Install [.net](https://dotnet.microsoft.com/en-us/download)
* * Clone the Repository: `git clone https://github.com/vietluu/TechHouseBTL.git`
